# Edgar Rodriguez Portfolio

Portafolio profesional de Edgar Rodriguez, desarrollador Backend / Fullstack Jr, construido con React, Vite, Tailwind CSS y Framer Motion.

## Objetivo

Mostrar experiencia real en desarrollo web, sistemas internos, automatización, integración de herramientas y proyectos aplicados a negocio.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- React Icons

## Secciones

- Hero profesional
- Experiencia
- Proyectos reales
- Stack tecnológico
- Contacto

## Proyectos destacados

- Hotel San Carlos POS
- SapiensAds AI
- Desarrollo Web Corporativo

## Instalación

npm install
npm run dev

## Build

npm run build
